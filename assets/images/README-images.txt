COLOCA AQUÍ LAS IMÁGENES DE PERFIL
====================================

Archivos necesarios (formato JPG, 16:7):

1. profile-explorador.jpg
2. profile-competitivo.jpg
3. profile-creativo.jpg
4. profile-narrativo.jpg

Tamaño recomendado: 1920 x 840 px (16:7)
Calidad: 85-90% JPG
Peso ideal: 200-400 KB cada una

Sugerencias de imágenes (libres de derechos):
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Pixabay (pixabay.com)

La imagen se muestra con un degradado oscuro en la
parte inferior para que el texto se lea bien.
