Video assets for the editorial sections
======================================

Place the final video files in this folder using these names:

- world-evolution.mp4
- reactive-systems.mp4

Optional WebM versions may also be added later for better browser optimization:

- world-evolution.webm
- reactive-systems.webm

Required poster images belong in assets/images/:

- video-world-evolution-poster.jpg
- video-reactive-systems-poster.jpg

Recommended specs
-----------------

- Format: MP4, H.264
- Duration: 8 to 18 seconds
- Audio: none or muted-safe
- Resolution: 1920x1080 or 1600x900
- File size target: below 8 MB per video when possible
- Motion style: atmospheric, abstract, cinematic, not distracting

Usage rules
-----------

- Videos should be decorative and support the narrative.
- Do not rely on audio for meaning.
- Avoid copyrighted videogame footage unless you have rights to use it.
- Prefer abstract, original, atmospheric or licensed stock footage.
- The HTML should include posters so the layout still works before videos load.
